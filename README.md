## 🔷 Overview  

This repository contains **lab work and assignments** completed during the **Computer Organization & Assembly Language (COAL)** course in the **3rd semester**.

It focuses on **low-level programming**, demonstrating how software interacts directly with hardware using **x86 Assembly Language**.

---

## 📂 Repository Contents  

---

### 🟣 🧪 Lab Work  

- Weekly lab exercises covering fundamental COAL concepts  
- Hands-on practice with assembly instructions and registers  
- Implementation of low-level programs using MASM  

---

### 🔵 📝 Assignments  

- Structured programming tasks based on course topics  
- Focus on logic building in assembly language  
- Problem-solving using low-level instructions  

---

## 🧠 Concepts Covered  

- x86 Assembly Language (MASM)  
- Registers & Instruction Set  
- Memory Addressing Modes  
- Stack Operations  
- Procedures & Macros  
- Input/Output Handling (Irvine32)  
- Debugging & Optimization  

---

## 🛠️ Tech Stack  

- 🧩 MASM (Microsoft Macro Assembler)  
- 📚 Irvine32 Library  
- 🖥️ Visual Studio  
- ⚙️ x86 Architecture  

---

## ▶️ How to Run  

### 🧑‍💻 Visual Studio  

1. Install **MASM (Microsoft Macro Assembler)**  
2. Setup **Irvine32 library**  
3. Open project in **Visual Studio**  
4. Add `.asm` file to project  
5. Build and run
